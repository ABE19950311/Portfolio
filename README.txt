・アプリ名
行政手続き一目アプリ（仮）

・サービスの概要
様々な数存在する行政手続きを行う上で下記内容が分かります
・手続きの際必要なもの
・具体的な流れ
・最寄りの市役所・銀行等の場所

・対象ユーザ
行政手続きに不慣れなユーザ

・ユーザの課題
日常生活を送る上で行政手続きを行う必要が度々訪れると思います。
人にもよると思うのですが私は頻度があまり多くなく、どこで何を行えば良いのか、手続きの際に必要なものは何か
具体的な手順はどうなるのかが分からず、インターネットで都度調べていたのですが、情報収集に時間を取られます。

・このアプリでその課題がどう解決されるか
様々な手続きが一つのアプリ上かつシンプルな操作で把握出来れば時間もあまりかからず、便利ではないかと思います
